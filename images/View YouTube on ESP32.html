1. If you mean watch YouTube videos on an ESP32

Unfortunately, you cant directly watch YouTube videos on an ESP32, because:

  • The ESP32 has very limited RAM (typically 520 KB) and no GPU for video decoding.

  • YouTube streams use H.264 or VP9, which are far too heavy for the ESP32 to decode in real time.

  • Even with an attached screen (like a TFT or OLED), the frame rate and resolution would be extremely low.

However, you can do simplified or related projects:

You can use the YouTube Data API to fetch video information via HTTP requests.
Example:

  • Displays video title, channel name, or thumbnail (converted to low-res bitmap) on a small screen (like a 1.8 TFT or OLED).

Libraries to use:

Your ESP32 can act as a remote control for YouTube on a PC or Chromecast:

#include <HTTPClient.h>
#include <ArduinoJson.h>

const char* ssid = "YourWiFi";

const char* apiKey = "YOUR_YOUTUBE_API_KEY";
const char* videoId = "VIDEO_ID";

void setup() {
  Serial.begin(115200);
  WiFi.begin(ssid, password);
  while (WiFi.status() != WL_CONNECTED) delay(500);

  String url = "https://www.googleapis.com/youtube/v3/videos?part=snippet&id=" + String(videoId) + "&key=" + String(apiKey);
  http.begin(url);
  int httpCode = http.GET();

  if (httpCode > 0) {
    String payload = http.getString();
    Serial.println(payload);
    // Parse JSON to get title, channel, etc.
  }
  http.end();
}

Option A Control YouTube via Bluetooth (ESP32 as Media Keyboard)

Requirements

  • ESP32 board (any variant)

  • Arduino IDE

  • Library: ESP32 BLE Keyboard by T-vk

Install the Library

  1. Open Arduino IDE Sketch Include Library Manage Libraries...

  2. Search for ESP32 BLE Keyboard Install

Example Code

cpp
#include <BleKeyboard.h>

BleKeyboard bleKeyboard("ESP32 YouTube Remote");

const int playPauseBtn = 12;  // connect button to GPIO12
const int nextBtn = 14;       // connect button to GPIO14

void setup() {
  pinMode(playPauseBtn, INPUT_PULLUP);
  pinMode(nextBtn, INPUT_PULLUP);
  bleKeyboard.begin();
  Serial.begin(115200);
}

void loop() {
  if (bleKeyboard.isConnected()) {
    if (digitalRead(playPauseBtn) == LOW) {
      Serial.println("Play/Pause");
      bleKeyboard.write(KEY_MEDIA_PLAY_PAUSE);
      delay(500);
    }
    if (digitalRead(nextBtn) == LOW) {
      Serial.println("Next Track");
      bleKeyboard.write(KEY_MEDIA_NEXT_TRACK);
      delay(500);
    }
  }
}

How It Works

  • The ESP32 acts as a Bluetooth Media Keyboard.

  • When you press a button, it sends a Play/Pause or Next command.

  • Your phone or computer interprets these just like normal keyboard media keys.

Works with:

  • YouTube in Chrome / Edge

  • YouTube mobile app (Android)

  • Spotify, Netflix, etc.

Option B Web Control (PC + Local Server)

If you prefer to control YouTube on a PC, you can:

  2. ESP32 sends HTTP commands (like /pause, /next) to that server.

  3. The server uses a browser automation library (e.g. pyautogui or selenium) to press keys in the YouTube tab.

This version lets you use Wi-Fi instead of Bluetooth good if your PC is far from the ESP32.

Step 1 PC Script (Python)

python
from flask import Flask, request
import pyautogui

app = Flask(__name__)

@app.route('/')
def index():
    return "YouTube Control Server is Running!"

@app.route('/playpause')
def play_pause():
    pyautogui.press('space')  # YouTube uses space for play/pause
    return "Toggled Play/Pause"

@app.route('/next')
def next_video():
    pyautogui.hotkey('shift', 'n')
    return "Next video"

@app.route('/back')
def previous_video():
    pyautogui.hotkey('shift', 'p')
    return "Previous video"

@app.route('/volumeup')
def volume_up():
    pyautogui.press('up')
    return "Volume up"

@app.route('/volumedown')
def volume_down():
    pyautogui.press('down')
    return "Volume down"

if __name__ == '__main__':
    app.run(host='0.0.0.0', port=5000)

Step 2 ESP32 Code (Arduino)

This code will send HTTP requests to your PC whenever you press a button.

cpp
#include <WiFi.h>
#include <HTTPClient.h>

const char* ssid = "YourWiFiName";
const char* password = "YourWiFiPassword";
const char* serverIP = "192.168.x.x";  // Replace with your PC's IP address

const int playPauseBtn = 12;
const int nextBtn = 14;
const int volUpBtn = 27;
const int volDownBtn = 26;

void sendCommand(const char* command) {
  if (WiFi.status() == WL_CONNECTED) {
    HTTPClient http;
    String url = "http://" + String(serverIP) + ":5000/" + String(command);
    http.begin(url);
    int httpCode = http.GET();
    http.end();
    Serial.printf("Sent command: %s -> %d\n", command, httpCode);
  }
}

void setup() {
  Serial.begin(115200);
  pinMode(playPauseBtn, INPUT_PULLUP);
  pinMode(nextBtn, INPUT_PULLUP);
  pinMode(volUpBtn, INPUT_PULLUP);
  pinMode(volDownBtn, INPUT_PULLUP);

  WiFi.begin(ssid, password);
  Serial.print("Connecting to WiFi");
  while (WiFi.status() != WL_CONNECTED) {
    delay(500);
    Serial.print(".");
  }
  Serial.println("\nConnected!");
}

void loop() {
  if (digitalRead(playPauseBtn) == LOW) {
    sendCommand("playpause");
    delay(500);
  }
  if (digitalRead(nextBtn) == LOW) {
    sendCommand("next");
    delay(500);
  }
  if (digitalRead(volUpBtn) == LOW) {
    sendCommand("volumeup");
    delay(500);
  }
  if (digitalRead(volDownBtn) == LOW) {
    sendCommand("volumedown");
    delay(500);
  }
}

  3. When you press a button, ESP32 will send an HTTP request to your PC.
